Explain what a token is used for.

Tokens are used so that servers can verify that the user is authenticated to use the services they provide & access their data. 


What steps can you take in your web apps to keep your data secure?

Use an SSL certificate. Make sure the site is using https instead of http. Require users to provide log-in credentials. Private routes 
Axios with Auth

Describe how web servers work.

A website will send a request to a physical webserver which checks the source and any authentification requirements and will respond with either
the data requested or an error message
